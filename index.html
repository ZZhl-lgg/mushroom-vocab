INSTALL APP

安装到手机桌面

iPhone / iPad

  1. 使用 Safari 打开本网页
  2. 点击底部的“分享”按钮
  3. 选择“添加到主屏幕”
  4. 点击右上角“添加”

在浏览器菜单中安装

打开浏览器菜单，选择“安装应用”“添加到主屏幕”或类似选项。

安装后会以独立窗口运行；词库和学习记录保存在当前设备浏览器中，请定期导出备份。
